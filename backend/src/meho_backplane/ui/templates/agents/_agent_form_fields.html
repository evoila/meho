Context contract (set by :func:`~meho_backplane.ui.routes.agents.forms._build_form_context`): * ``model_tiers`` -- list[str] of AgentModelTier enum values. * ``errors`` -- dict[field -> message]; empty on a fresh render, populated on a 409 / 422 re-render. * ``values`` -- dict[field -> submitted/pre-populated value]; echoed back so the operator keeps their input. * ``mode`` -- "create" | "edit". In edit mode the ``name`` input is read-only (name is the per-tenant natural key, not updatable). ``identity_ref`` is a free-text field for the scaffold; the picker over registered non-revoked principals is T4 (#1832).
