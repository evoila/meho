{#- SPDX-License-Identifier: Apache-2.0 Copyright (c) 2026 evoila Group Audit row detail drawer "not found" fragment (Initiative #1841 G10.15, Task #1945). Returned with HTTP 404 by ``GET /ui/audit/show/{audit_id}`` when the requested id does not resolve in the operator's tenant. The boundary is opaque: a cross-tenant id and a non-existent id both render this fragment at 404 (never a 403, which would leak that the id exists for another tenant). Mirrors ``broadcast/_event_drawer_not_found.html``. It carries the same ``x-data``/``click.outside`` dismiss island so a click outside (or Escape / Close) clears it consistently. -#}

Audit row not found

No audit record with id {{ audit_id }} exists in this tenant.

The record may have aged out of the retention window, or the id may belong to another tenant (the boundary is opaque).
