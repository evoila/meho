Dismiss model (Alpine click-outside) ------------------------------------ The ``aside`` carries its own Alpine island: ``x-data="{ open: true }"`` + ``x-show="open"`` so it is visible on swap-in, and ``x-on:click.outside="open = false"`` so a click anywhere outside the drawer dismisses it. Alpine only evaluates ``.outside`` while the element is visible, so the swap-in click that opened the drawer cannot immediately re-close it (the documented race the ``.outside`` helper guards against). A Close button mirrors the same ``open = false`` for keyboard / explicit dismissal. Sections: 1. Header -- op_id + op_class badge + close. 2. Operation -- operator, method/path, status, timing. 3. Identifiers -- request_id, audit_id, broadcast event_id. 4.
